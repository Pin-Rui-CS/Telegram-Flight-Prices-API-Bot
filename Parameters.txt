
FLIGHT OFFERS SEARCH API

Mandatory Parameters:
| Parameter                 | Type                | Description                                                     |
| ------------------------- | ------------------- | --------------------------------------------------------------- |
| `originLocationCode`      | string              | IATA code of departure city/airport (`SIN`, `TYO`, `LON`, etc.) |
| `destinationLocationCode` | string              | IATA code of arrival city/airport                               |
| `departureDate`           | string (YYYY-MM-DD) | Departure date                                                  |
| `adults`                  | integer             | Number of adult passengers                                      |

Common Optional Parameters:
| Parameter              | Type                     | Description                                                    |
| ---------------------- | ------------------------ | -------------------------------------------------------------- |
| `returnDate`           | string (YYYY-MM-DD)      | Date of return (for round trip)                                |
| `children`             | integer                  | Number of children passengers (2–11 yrs)                       |
| `infants`              | integer                  | Number of infants (under 2 yrs)                                |
| `travelClass`          | string                   | Cabin class: `ECONOMY`, `PREMIUM_ECONOMY`, `BUSINESS`, `FIRST` |
| `includedAirlineCodes` | string (comma-separated) | Restrict to certain airlines (`SQ`, `JL`, `BA`, etc.)          |
| `excludedAirlineCodes` | string (comma-separated) | Exclude certain airlines                                       |
| `nonStop`              | boolean/string           | `"true"` or `"false"` → filter direct flights only             |
| `currencyCode`         | string                   | Force price currency (e.g., `SGD`, `USD`, `EUR`)               |
| `maxPrice`             | integer                  | Max price filter (in chosen currency)                          |
| `max`                  | integer                  | Max number of flight offers returned (sandbox default: 250)    |
| `departureDateWindow`  | integer                  | +/- days from departure date                                   |
| `returnDateWindow`     | integer                  | +/- days from return date                                      |
